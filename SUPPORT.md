# Support

Use the public repository channels for general help so other students can learn from the answer.

## Questions and Project Help

- Use GitHub Issues for reproducible bugs, broken commands, missing files, or documentation problems.
- Use GitHub Discussions if they are enabled for broader questions, learning advice, or project selection help.
- Include the project path, operating system, tool versions, command run, expected result, actual result, and sanitized logs.

## Security Reports

Do not open a public issue for vulnerabilities, exposed credentials, or unsafe deployment defaults. Follow `SECURITY.md` and report privately through GitHub security advisories or the listed maintainer contact.

## What Not To Share

Do not post private credentials, cloud account IDs, access keys, kubeconfigs, `.env` files, `*.tfvars`, Terraform state, private keys, or screenshots that expose secrets.

Before sharing logs, remove tokens, passwords, account numbers, hostnames that should remain private, and any personally identifying data.

## Scope

This is a learning repository. Maintainers can help with documentation, reproducibility, validation, security hygiene, and project structure. They cannot operate your cloud account, debug private infrastructure directly, or recover leaked credentials for you.
