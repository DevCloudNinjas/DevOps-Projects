A guided program for learners who want to finish portfolio-ready DevOps labs
with review, accountability, and a clearer path from project work to career
proof.

The free portal teaches the labs. The Accelerator helps students finish the
right labs, avoid expensive mistakes, explain their work clearly, and leave with
reviewable GitHub evidence.

## Program Structure

| Week | Focus | Student Output |
|---|---|---|
| 1 | Setup, safety, and path selection | Selected route, local validation, cost-risk plan |
| 2 | First deployable or local lab | Working project, command proof, notes |
| 3 | CI/CD, IaC, or Kubernetes lab | Pipeline, plan/apply, cluster, or GitOps evidence |
| 4 | DevSecOps or observability proof | Scan output, metrics/traces, policy, or cleanup proof |
| 5 | Portfolio polish | README, architecture summary, screenshots |
| 6 | Review and interview prep | Final case study, talking points, next project plan |

## What Is Included

- Weekly group session or office hour.
- Project selection help.
- Cost and credential safety review before cloud work.
- GitHub README and portfolio feedback.
- Troubleshooting support for blocked labs.
- Cleanup evidence review.
- Interview explanation prompts.
- Premium Kit access.

## What Is Not Included

- No access to student cloud credentials.
- No promise of job placement.
- No running paid cloud resources on behalf of students.
- No private managed infrastructure service.
- No shortcut around actually doing the labs.

## Application Questions

Use these questions before accepting someone into the Accelerator:

1. What DevOps level are you at today?
2. Which route do you want: beginner, Docker/Kubernetes, AWS/Terraform, DevSecOps, or portfolio?
3. Do you have a local machine that can run Docker?
4. Are you willing to use your own cloud account only after reading the safety guide?
5. What do you want to show at the end: GitHub project, LinkedIn post, resume bullet, or interview story?
