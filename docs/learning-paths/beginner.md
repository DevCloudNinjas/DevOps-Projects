# Beginner Learning Path

This path is for students who need a confident first pass through Linux, app packaging, CI/CD, cloud vocabulary, and basic security.

## Sequence

| Step | Project | Why it comes here |
|---|---|---|
| 1 | [03 - Linux Fundamentals](https://github.com/DevCloudNinjas/DevOps-Projects/tree/master/project-03-linux-fundamentals) | Start with shell, files, permissions, and operational basics. |
| 2 | [49 - Text Encryption Cybersecurity Demo](https://github.com/DevCloudNinjas/DevOps-Projects/tree/master/project-49-text-encryption-cybersecurity) | Learn a small local security concept before touching cloud resources. |
| 3 | [04 - Django Application on AWS ECS and ECR](https://github.com/DevCloudNinjas/DevOps-Projects/tree/master/project-04-django-aws-ecs) | Package an app with Docker and understand container delivery. |
| 4 | [10 - .NET CI/CD with Azure App Service](https://github.com/DevCloudNinjas/DevOps-Projects/tree/master/project-10-dotnet-azure-devops) | Study a simpler managed-platform CI/CD flow. |
| 5 | [14 - Android CI/CD with GitHub Actions](https://github.com/DevCloudNinjas/DevOps-Projects/tree/master/project-14-github-actions-android) | Practice a low-cost GitHub Actions workflow. |
| 6 | [50 - ArgoCD GitOps Home Lab](https://github.com/DevCloudNinjas/DevOps-Projects/tree/master/project-50-argocd-gitops-home-lab) | Finish with a local modern delivery loop. |
