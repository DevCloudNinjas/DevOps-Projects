# DevOps Projects Documentation

This site collects the repository's reusable learning and operations material:

- [Catalog](catalog/index.md): how to navigate projects, learning tracks, and reusable resources.
- [Runbooks](runbooks/index.md): repeatable operating procedures for local validation and lab maintenance.
- [Flagship docs](flagship/index.md): high-signal project guides worth treating as primary examples.
- [Security baselines](security-baselines/index.md): reusable checklists for Kubernetes, Terraform, Dockerfile, CI/CD, and secrets work.
- [Marketing kit](marketing/index.md): positioning, LinkedIn posts, and case-study templates for sharing the portfolio.

Use these docs as the default entry point before changing infrastructure, pipeline, or deployment examples.
