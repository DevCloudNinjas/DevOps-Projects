# DevOps Portfolio Premium Kit

The Premium Kit is the paid self-serve companion for students who want to turn
lab work into polished portfolio proof faster.

The public labs remain free. The kit packages the repeatable assets students
usually have to invent from scratch: tracking sheets, proof checklists, README
templates, architecture prompts, cleanup logs, and interview explanation notes.

## Suggested Pricing

Start simple:

| Tier | Price | Best For |
|---|---:|---|
| Student Kit | $29 | Templates, checklists, workbook, proof guide |
| Portfolio Kit | $49 | Student Kit plus diagram prompts and interview sheets |
| Team Kit | $149 | Small study groups, mentors, and bootcamp assistants |

Pricing can change after validation. The first version should prove whether
students actually want structured help finishing and presenting the labs.

## Delivery Plan

1. Publish the free portal first.
2. Create the kit as a PDF or ZIP bundle.
3. Sell it through a simple checkout platform.
4. Add the checkout link here only after the product exists.
5. Update the kit monthly with new templates, lab notes, and fixes.

## Ethical Boundary

The Premium Kit should not sell access to other people’s code. It should sell
curation, workflow, documentation structure, diagrams, checklists, and review
assets created for this portal.

## Best First Kit Version

Ship a focused first version:

- One lab planning workbook.
- One project README template.
- One portfolio case-study template.
- One cleanup checklist.
- One screenshot/proof checklist.
- Five interview explanation worksheets for the modern 2026 labs.

After that, improve it based on what learners actually ask for.
