## Badge Reference

| Badge | Meaning |
|---|---|
| `beginner-friendly` | Good first project with low setup complexity. |
| `local-first` | Can be completed mostly on a local machine. |
| `free-tier-aware` | Designed to fit free-tier or minimal cloud usage when followed carefully. |
| `cloud-cost-risk` | Requires extra budget awareness and cleanup discipline. |
| `portfolio-ready` | Produces a strong demo, runbook, or case-study artifact. |
| `team-workflow` | Practices collaboration, review, or delivery workflow skills. |
| `security-focused` | Emphasizes scanning, hardening, secrets, or supply-chain controls. |
| `needs-cloud-account` | Requires a real cloud provider account. |
| `cleanup-required` | Creates resources that should be destroyed after the lab. |
| `capstone` | Best attempted after completing related foundation projects. |
